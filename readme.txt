Add InviteBox-powered referral program to your WordPress site and promote it virally.

== Description ==

Reach new customers today with a social referral program. Design and launch <a href="http://invitebox.com/">refer a friend</a> promotions on your WordPress website or blog in minutes.

You can set up one of the following types of referral campaigns:

1. Referral program with instant rewards.

2. Referral contest.

3. Referral program with goal-based rewards.

and reward your customers for sharing your site with their friends. Learn more about InviteBox-powered <a href="http://invitebox.com/how-it-works/">referral programs</a>.

== Installation ==

1. Visit <a href="http://invitebox.com/pricing" title="InviteBox website">InviteBox website</a> and sign up for a plan that suits you best.

2. Create your referral campaign, set the rules and setup your InviteBox widget.

3. Copy your widget 'Secret key' from the 'API settings' section on the itegration page.

4. Unzip and Upload file wordpress-invitebox-plugin.php to the /wp-content/plugins/ directory

5. Activate the plugin through the 'Plugins' menu in WordPress using your personal widget key.

6. InviteBox plugin will be automatically added to every page on your side.
